# SQL-CRIME
Montgomery County Police Department Record Of Crime Organized through SQL 

##  Project Overview
This project focuses on organizing and analyzing crime data from Montgomery Country, Maryland.
The dataset includes records of reported crimes such as offense type, date, time, location, police district,
and case status.

The goal of this project is to transform raw crime data into a structured relational database that allows users
to query and analyze trends efficiently.

##  Dataset Information
- Source: Montgomery County Open Data Portal
- Time Range: May 1, 2016 - May 1, 2026
- Records: 300,000+ crime incidents
- Format: CSV imported into MYSQL
